# wvlet-lib

Action builder for [wvlet-lib](https://github.com/wvlet/wvlet/tree/main/wvc-lib)


### Usage
#### vcpkg
```json
{
  "dependencies": [
    "wvlet-lib"
  ],
  "overrides": [
    {
      "name": "wvlet-lib",
      "version-string": "0.0.1",
      "repository": "https://github.com/quackscience/wvlet-lib.git"
    }
  ]
}
```
